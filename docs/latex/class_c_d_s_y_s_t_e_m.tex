\doxysubsection{Detailed Description}
The \doxylink{class_c_d_s_y_s_t_e_m}{CDSYSTEM} class manages the operations for CD-\/\+ROMs using different file systems. 

This class provides functionality for handling ejected and inserted CDs, burning CDs, processing files and folders, and managing file systems like FAT and INODE.
